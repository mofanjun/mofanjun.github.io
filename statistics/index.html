# 统计学

统计学是收集 处理 分析 解释数据并从数据中得出结论的科学。 按照统计方法分，可以分为：

# 描述统计方法

研究的是数据收集 处理 汇总 图表描述 概括 与分析等统计方法。

# 推断统计方法

利用样本数据推断总体特征的统计方法。

# 数据统计类型

# 按测量尺度划分

  • 分类数据-只能归类于某一类别的非数字型数据，它是对事物进行分类的结果，数据表现为类别，只能用文字来表述。
  • 顺序数据-只能归类于某一有序类别的非数字型数据。
  • 数值型数据-是按数字尺度测量的观察值 其结果表现为具体的数值。

# 按收集方式分析

  • 观测数据
  • 实验数据

# 按照现象与时间的关系分

  • 截面数据
  • 时间序列数据

提示

区分数据类型十分重要，因为对不同的类型的数据需要采用不同的统计方法来处理和分析。

# 基本概念

# 总体

是包含所研究的全部个体（数据）的集合，分为：

  • 有限总体
  • 无限总体

# 参数

用来描述总体特征的概括性数字度量，也就是总体的某种特征值:

  • -总体平均数
  • -总体标准差
  • -总体比例

# 统计量

  • -样本平均数
  • S-样本标准差
  • P-样本比例

# 变量

  • 分类变量-说明实务类别的一个名称，其取值是分类数据。
  • 顺序变量-说明事物有序类别的一个名称，其取值是顺序数据。
  • 数值型变量
    • 离散型变量-只能取有限个值 可以一一列举 只能取可数值的变量。
    • 连续性变量-取值是连续不断的 不能一一列举 在一个或多个区间中取任何值的变量。
